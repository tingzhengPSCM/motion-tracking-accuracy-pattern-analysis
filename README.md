# motion-tracking-accuracy-pattern-analysis

This repository contains the R scripts used for the statistical analysis and figure generation in the manuscript.

The scripts include:

- GAM fitting and model comparison
- residual diagnostics
- figure generation
- RULA posture risk scoring


## Data availability

The raw participant-level data are not publicly shared due to privacy and ethical considerations.

## Software environment

The analysis was conducted in R.
